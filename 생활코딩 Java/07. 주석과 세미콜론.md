### 주석
```java
public static void main(String[] args){

	// 한 줄 주석
	// 두 개의 변수가 같은 데이터 타입일 때 아래와 같이 코드를 작성한다.
	String a, b;

	/* 여러 줄 주석
	a = "coding";
	b = "everybody";
	System.out.println(a+b);
	*/
}

- 자바의 문서를 만들 때 사용한다.
**/


```

### 세미콜론
- 문장의 끝을 의미
- 사용하지 않으면 컴파일 에러가 발생
- 세미콜론을 이용하여 여러 개의 문장을 한 줄에 표현할 수 있다.
